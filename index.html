دورة HTML

وصف الدورة

هي لغة ترميز أساسية تستخدم لإنشاء صفحات الويب. تقوم HTML بتنظيم وترتيب المحتوى داخل الصفحة، مثل النصوص، الصور، العناوين، الجداول، والروابط. تعتمد HTML على عناصر تُسمى “Tags” (علامات) تُحيط بالمحتوى لتحدد نوعه ووظيفته. كل صفحة ويب تبدأ بملف HTML يحتوي على هيكل الصفحة الكامل. يمكن للعناصر أن تحتوي على سمات (Attributes) تضيف خصائص إضافية مثل الألوان أو الروابط. HTML ليست لغة برمجة، بل لغة توصيف تُخبر المتصفح كيف يعرض المحتوى. يُستخدم معها لغات أخرى مثل CSS لتنسيق التصميم وجعله جذابًا، وJavaScript لإضافة التفاعل والوظائف. تتطور HTML عبر نسخ متعددة، وأحدثها HTML5 التي أضافت ميزات حديثة مثل الفيديو والصوت والتخزين المحلي. تُعتبر HTML حجر الأساس في تصميم الويب، وفهمها ضروري لأي مطور ويب. يمكن كتابة ملفات HTML باستخدام أي محرر نصوص بسيط، ثم عرضها عبر أي متصفح إنترنت. تعلم HTML هو أول خطوة لأي شخص يرغب في الدخول إلى مجال تطوير المواقع والتطبيقات على الإنترنت. باختصار، HTML هي اللغة التي تُبنى بها صفحات الويب وتُعرض محتوياتها للمستخدمين.
